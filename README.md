# Products Management API (CRUD)

## About
A simple and professional application to manage products.  
You can **create, update, delete, and view products** in a clean interface.  
Each product has a **title, price, taxes, ads, discount, total, count, and category**.  
When editing a product, the form scrolls to the top automatically for a smooth user experience.  

What are you waiting for? Start managing your products easily and efficiently!
